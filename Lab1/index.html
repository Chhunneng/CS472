Block vs. Inline HTML Elements

Block Elements

  • <div>
  • <p>
  • <h1> - <h6>
  • <hr>
  • <section>
  • <header>
  • <footer>
  • <nav>
  • <aside>
  • <article>
  • <ol> / <li>
  • <ul> / <li>
  • <form>

Inline Elements

  • <span>
  • <br>
  • <img>
  • <a>
  • <strong>
  • <i>
  • <input>
  • <textarea>
  • <button>
  • <select> / <option>
  • <label>
